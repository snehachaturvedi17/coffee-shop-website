# Coffee Shop Website ☕

## 📖 Overview
This project is a simple and stylish Coffee Shop website created using HTML and CSS.  
It represents a modern café landing page with a warm maroon and brown aesthetic theme.

The website showcases a coffee shop’s atmosphere, menu highlights, and design layout.

## ✨ Features
- Elegant coffee shop themed design
- Hero section with café background
- Menu cards displaying coffee and desserts
- Clean and modern layout
- Hover effects using CSS
- Responsive design structure



## 🛠 Technologies Used
- HTML5
- CSS3
